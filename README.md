# 🔥 Firebase SDK Integration Check

## 📌 Назначение
Тестовый проект для проверки корректности подключения Firebase SDK к iOS приложению.

## ✅ Что реализовано
- Инициализация FirebaseApp в AppDelegate
- Регистрация событий Analytics
- Базовая конфигурация GoogleService-Info.plist

## 🗓 Дата создания
Март 2026

## 📝 Примечание
Проект сохранён как рабочий пример.
Используется как референс для будущих проектов с Firebase.
