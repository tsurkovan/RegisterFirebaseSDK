# 🔥 Sign up / Register screen

## 📌 Назначение
Приложение "Экран авторизации" с использованием библиотеки Firebase SDK и фреймворка SwiftUI.

## ✅ Что реализовано
- Регистрация нового пользователя
- Вход в аккаунт
- Отображение email на главном экране
- Кнопка выхода

Март 2026

Используется как референс для будущих проектов с Firebase.
